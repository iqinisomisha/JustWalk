# JustWalk3

This repo contains the JustWalk3 SwiftUI project. Custom fonts are stored in the `Fonts` directory and included in the Xcode project.

## Custom Font

`PatrickHand-Regular.ttf` is bundled with the application. The font file resides under `JustWalk3/Fonts/PatrickHand-Regular.ttf` and is referenced by the Xcode project. If you replace this file with the official font, clean and rebuild the app so SwiftUI can locate it.
